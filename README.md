# \# Monitoring Ruangan Arduino

# 

# Arduino-based room monitoring system using DHT11, LDR, RGB LED, buzzer, and LCD I2C to monitor temperature, humidity, and light intensity in real-time.

# 

# \## Features

# \- Temperature monitoring

# \- Humidity monitoring

# \- Light intensity monitoring

# \- RGB LED temperature indicator

# \- Humidity warning buzzer

# \- LCD I2C display output

# 

# \## Components

# \- Arduino Uno

# \- DHT11

# \- LDR Sensor

# \- LCD 16x2 I2C

# \- RGB LED

# \- Active Buzzer

# 

# \## Libraries

# \- DHT.h

# \- Wire.h

# \- LiquidCrystal\_I2C.h

# 

# \## Project Structure

# ```text

# code/          -> Arduino source code

# docs/          -> Documentation and reports

# images/        -> Project images

# components/    -> Component information

# ```
